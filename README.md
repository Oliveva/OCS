# 信号编码子系统

- ### 类型

    | 编码|场景|
    |:---:|:---:|
    |纠错编码|抗信道噪声|
    |压缩编码|卫星带宽资源有限（遥感）|
    |加密编码|军事、商业敏感信息|
    |同步编码|卫星与地面时钟差异|
    |交织编码|突发性干扰|

    - #### 纠错

        卷积码

        turbo、Reed-Soloman

        Viterbi解码

    - #### 交织

        矩阵块
